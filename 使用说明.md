# EyeMotion 使用说明

## 程序简介
EyeMotion 是一款眼部运动训练程序，通过追踪屏幕上移动的小球来进行眼部肌肉锻炼。

## 操作说明

### 键盘控制
- **空格键 (SPACE)**：开始游戏 / 暂停/继续
- **P键**：开启/关闭背景音乐
- **R键**：游戏结束后重新开始
- **ESC键**：退出程序

### 窗口控制
- **右上角按钮**（从左到右）：
  - 蓝色按钮：最小化窗口
  - 黄色按钮：全屏/窗口切换
  - 红色按钮：关闭程序

## 训练关卡
- **LEVEL 1-4**：不同速度和角度的线性运动
- **LEVEL R**：圆周运动训练

每个关卡时长：45秒

## 医学益处分析
根据最新的眼科医学研究与视光训练理论，EyeMotion 所提供的“动态视觉追踪”训练（Visual Tracking Exercises）对眼部健康及神经系统具有多重积极意义：

### 1. 强化平滑尾随运动 (Smooth Pursuit)
- **原理**：训练眼球在保持头部不动的情况下，连续、流畅地追踪移动目标的能力。
- **益处**：增强眼外肌精细控制能力，提升双眼协调运作效率，改善阅读跳行、漏字等问题。

### 2. 缓解数码视疲劳 (Digital Eye Strain)
- **原理**：打破长时间注视静止目标（如文档、网页）导致的调节痉挛。
- **益处**：促进眼部血液循环，通过大范围运动放松紧绷的肌肉群。

### 3. 提升视觉专注力与神经连接 (Neuro-Visual Processing)
- **原理**：追踪快速移动目标需要“眼-脑”毫秒级的配合。
- **益处**：提升大脑处理视觉信息的速度和准确性，激活大脑注意力网络，增强专注力。

### 4. 辅助动眼神经功能复健
- **适用性**：改善注视不稳，提升在球类运动或驾驶场景下的动态视觉捕捉能力。

## 系统要求
- Windows 10/11
- 显示器分辨率：建议 1600x900 或更高

## 🛡️ 安全运行指南
由于本程序是开源项目，未购买昂贵的 Windows 数字证书，因此在您从 GitHub 下载并运行时，Windows SmartScreen 可能会拦截并显示“Windows 已保护你的电脑”警告。

### 解决方法：
1. 在弹出的蓝色窗口中点击 **“更多信息”**。
2. 点击右下角出现的 **“仍要运行”** 按钮。

或者：
1. 右键点击 `eyemotion.exe`，选择 **“属性”**。
2. 在底部的“安全”栏目勾选 **“解除锁定”**。
3. 点击“确定”，再次运行即可。

> *提示：本程序代码完全透明公开，您可以随时在 GitHub 仓库查看源码以确保安全。*
